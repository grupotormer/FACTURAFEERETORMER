Estructura de Base de Datos Vinculada (App Sheep / AppSheet)

Esta aplicación web sincroniza sus cambios directamente con el backend de datos de App Sheep. Se mantiene una relación estricta de uno-a-muchos (1:N) entre la cabecera y el detalle para preservar la integridad referencial requerida por la aplicación móvil y web.

Tabla: PREVENTAS

Contiene la cabecera, cliente, contacto, total acumulado y fecha de la preventa.

Tabla: DETALLE_DE_PREVENTAS

Contiene cada ítem, cantidad, precio unitario y referencia preventa_id (Foreign Key).

Vinculación Certificada

El esquema relacional SQLite mapea 100% de manera nativa con las tablas de Google Sheets utilizadas en AppSheet.
